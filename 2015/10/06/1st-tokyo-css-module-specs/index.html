第一回　W3C CSS Module 仕様書もくもく会@東京　の復習

10-06-2015
event

コレを書いたのがきっかけで、@kuboshoさんがCSS仕様書もくもく会のコミュニティをつくってくださいました。
このとき私は、ブログの投稿から@kuboshoさんが見てくださって、コミュニティが立つまでの速さに大変驚き震えていたことを覚えています。
そして、「一応たかはしさんが言い出しっぺなので。」と、主催者としてお誘いいただきまして、2015年9月28日に第一回もくもく会を開催しました。
イベント主催どころかもくもく会に参加したことすらない私は、右も左もわからぬまま当日を迎え、@kubosho_さん及び参加者の方には大変ご迷惑をおかけいたしましたことをここにお詫び申し上げます。

  • @kubosho_さんトゥギャってくれました。　http://togetter.com/li/879946
  • 当日読んだ日本語訳 http://standards.mitsue.co.jp/resources/w3c/TR/css3-selectors/
